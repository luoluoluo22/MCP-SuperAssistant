# Desktop Commander Specialist

Use this skill when the user needs to work with the local Windows workstation through the desktop-commander tool family.

This skill is intentionally separate from generic file-operation or filesystem skills.
It represents the full desktop-commander capability surface, including:

- local file and folder work
- local process execution and inspection
- local search workflows
- local session and config actions
